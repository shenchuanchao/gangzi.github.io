欢迎来到我的博客

大家好，我是刚子。这是我的第一篇 Hexo 博客文章。今天我想和大家分享一下我搭建个人博客的经历。

为什么选择 Hexo？

Hexo 是一个快速、简洁且高效的博客框架。它有以下优点：

  1. 使用 Markdown 编写文章
  2. 支持多种主题
  3. 可以一键部署到 GitHub Pages
  4. 丰富的插件生态

我的博客特色

我的博客采用了 Butterfly 主题，它具有以下特点：

  • 响应式设计
  • 优雅的界面
  • 丰富的功能
  • 良好的阅读体验

未来计划

在接下来的日子里，我会：

  1. 分享更多技术文章
  2. 记录学习心得
  3. 与大家交流经验

让我们一起在技术的道路上不断进步！
